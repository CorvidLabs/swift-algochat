# Design

Add `EnvelopeSecurity` to the existing unit filter, increasing that batch from 155 to 169 tests and the deterministic total from 223 to 237. Keep the CLI gate noninteractive by building the `algochat` product without executing the interactive program.

Treat Trust, Fledge, Augur, Attest, managed-agent instructions, and installed agent command directories as meaningful SDD paths. Gemini create-change uses `{{args}}`. Every create-spec integration strips the optional flag first, treats the entire remaining value as a bare module name only when it contains no whitespace and only identifier characters, and otherwise derives a concise name from the full description.

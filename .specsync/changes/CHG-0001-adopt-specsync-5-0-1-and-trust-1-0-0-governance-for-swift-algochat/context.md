# Context

Swift AlgoChat is a Swift 6 protocol library and CLI with Linux and macOS verification, documentation publication, and tagged releases. Its cryptographic suite is deliberately split into CI-safe batches because excessive invocations and several integration suites are unstable or require a localnet. The repository has no prior SpecSync companions or coverage threshold.
